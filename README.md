# Guliston Texnopark

Guliston Yoshlar Texnoparki veb-ilovasi (React, TypeScript, Vite).

**Ishlab chiquvchi:** Sadullayev Faxriyor

**GitHub:** [https://github.com/fakhriyor21/Gulistantechnopark](https://github.com/fakhriyor21/Gulistantechnopark)

## O‘rnatish

```bash
npm install
```

## Ishlab chiqish serveri

```bash
npm run dev
```

## Production build

```bash
npm run build
```

## Muhit o‘zgaruvchilari

```env
VITE_API_ORIGIN=http://localhost:3000
```
